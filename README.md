# businessrifa

Sistema profissional de rifas online com painel administrativo, reserva de numeros, Pix copia e cola, integracao configuravel com InfinitePay, sorteio, area do cliente e preparo para deploy.

## Documentacao

- Manual do sistema: [README-BUSINESS-RIFA.md](README-BUSINESS-RIFA.md)
- Deploy em producao: [DEPLOY-PRODUCAO.md](DEPLOY-PRODUCAO.md)

## Comandos

```bat
npm.cmd install
npm.cmd run build
npm.cmd start
```

Teste automatico principal:

```bat
npm.cmd run test:smoke
npm.cmd run test:cleanup
```
